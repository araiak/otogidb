これは新規（または比較的新しい）プレイヤー向けのタワーレイドイベントガイドです。タワーイベントは新規プレイヤーにとって非常に良いです。ギルドメンバーに助けを求めることができるからです - 親切なギルドを見つければ、助けを借りて複数の5つ星カードを獲得できる可能性が高いです。

タワーイベントでは、通常コンテンツをプレイするか、SEを使ってボスを探すことでレイドボスに攻撃されます。

## クイックスタートチェックリスト

### イベント前
1. **アクティブで親切なギルドを見つける** - 難しいフロアで助けを得るために重要
2. **フレンドリストを確認**して :card[740] ヘルパーを探す - 必要ならファンタズマプレイヤーを追加（またはアニマ/ディビナキャリーを好むならそちらも）
3. **タンクを選ぶ** - 下のリストから（または持っている耐久カードを使用）
4. **リザーブを埋める** - 持っているダメージアシストやチームバッファーで
5. **タンクが必要なら**オークションを確認

### イベント中
1. **編成:** 借りたキャリー + 自分の耐久カード1枚（3-5 SE）
2. **タイマーまで生存** → ギルドにシェアして助けを求める
3. **全滅？** リバイブしない（カードドロップフロア以外）
4. **壁にぶつかった？** 自分のカードを追加するかタンクを強化してタイマーまで生存できるようにしてシェア

## タワーイベントの特徴

タワーイベントは：

- **複数ラウンド**なので、ほぼ全てのスキルとアビリティが有用：
  - *最終ウェーブアビリティ*は戦闘の最後のウェーブで発動
  - *累積ボーナス*は「ウェーブ開始時にXを獲得」- 複数ラウンドで積み重なる
  - *固定ボーナス*は常に有効
- **無属性**なので、属性に関係なく最強チームを使用
- **可変SEコスト**（カードに基づき3-18 SE）
- **継続可能** - タイムアウトしたら、SE+1でダメージを与え続けるか、ギルドにボスをシェアして助けを求められます。例えば、チームが18 SEなら、19 SEで同じダメージを再度与えられます。
- **XPとドロップ率ブーストは効果なし** - ストーリーイベント用に取っておきましょう！

### SEコスト計算

**重要:** SEコストにカウントされるのは自分のカードだけ。借りたフレンドとリザーブは無料！

SEコスト = 自分の1-4枚のカードの星の合計（最小3、最大18）

- 自分の1× 3つ星 = 3 SE
- 自分の1× 5つ星 = 5 SE
- 自分の2× 5つ星 = 10 SE
- 借りたフレンドのカード = 無料
- リザーブスロット = 無料

これにより、強力な借りたキャリーと自分の安いカードを非常に低いSEで使えます。

## 新規プレイヤー向けヒント

### 生存を優先！

タイマーまで生存すれば、助けを求めてボスをシェアするか、SEで再攻撃できます。全滅するとリバイブが必要で、高コストです。

**リバイブしない** - 重要なカードドロップフロア以外では！

**壁にぶつかっても大丈夫** - 助けを求めましょう！

## 低SE登頂戦略

借りたヘルパーは無料なので、最適な戦略は強力なダメージキャリーを借りて、自分のカードは1枚だけ用意することです。自分のカード1枚だけで可能な限り進めます - 3つ星は3 SE、5つ星は5 SEです。

**常にリザーブを埋めましょう！** リザーブはSEコストにカウントされないので、強力なアシストとサポートアビリティを詰め込みましょう。

**注意:** 以下のカードは、ほとんどのプレイヤーが合理的にアクセスできる例に過ぎません - 機能するカードはたくさんあります！これらは探すべきアクセスしやすいオプションの一部です。

### 投資優先順位

ヒーラーなしの戦闘ではダメージを追加することが非常に重要です - 早くクリアするほど、受けるダメージが減ります。優先順位は：

1. **十分に耐久力のあるタンクを入手** - 重要なタンクアビリティを解放するのに十分なコピーを購入（レベルは下記参照）
2. **リザーブとアシストに投資** - ダメージブーストとDRアシストは、タンクをさらに強化するより効果的
3. **タンクの最大化は最後** - MLBは良いですが、強力なリザーブより重要度は低い

**モチ最適化のヒント:** 重要なアビリティを解放するために必要な最小レベルに注意。リザーブやアシストカードは、アビリティが解放されたらシールして売り戻し、投資の一部を回収できます。

### ファンタズマセットアップ（推奨）

:card[740] がフレンドヘルパーとして広く利用可能で、耐久力のあるファンタズマカードが入手しやすいため、これが最もアクセスしやすいセットアップです。

**借りる:** :card[740] - フレンドから広く利用可能、高い攻撃力、高いスキルダメージ、累積ダメージブーストはあなたのカードにも影響。

**あなたのカード（優先順位順）:**

1. :card[738] (Lv. 80) - サポートアビリティが攻撃力を大幅にブースト
2. :card[227] (Lv. 70) - 良い自己回復
3. :card[1489] (Lv. 80) - 借りた酒呑とのスキルダメージシナジー、低HP時の自己回復
4. :card[1436] (Lv. 75) - 耐久力のある最近のイベントカード、入手しやすい
5. :card[267] (Lv. 75) - オークションから、バトル時間は弱いチームに役立つ、良いEHP（注意：ギルドにシェアするために生存しようとする場合、追加のバトル時間は不利に働く可能性あり）
6. :card[687] (Lv. 80) - オークションからの攻撃的オプション、DRアシストかヒーラーが早めに必要かも

### ディビナセットアップ

**あなたのカード（優先順位順）:**

### アニマセットアップ

アニマはアクセスしやすいオプションが少なく、キャリーはフレンドリストであまり見かけません - アニマプレイヤーを意識的に追加する必要があるかもしれません。

リザーブは無料なので、その価値を最大化：

- :card[664] (Lv. 85) - 2カードダメージブースト

- :card[643] (Lv. 75) - マルチターゲットダメージ

**クリティカルダメージブースト**は、リザーブに :card[1307] や :card[1494] があれば特に強力です。

**ダメージ軽減（DR）アシスト**は高レベルでタンクの生存を助けます。例：:card[1062]、:card[688]、:card[1437]、:card[749]、:card[657]、:card[677]

## チームのスケールアップ

壁にぶつかったら、自分のカードを段階的にチームに追加します。常に借りたフレンド1人がいて、SEコストは自分の1-4枚のカードの星の合計です（最小3、最大18）。

**2枚目のカードはおそらくヒーラーにすべき** - これでタイマーまで生存し、ギルドにシェアして助けを求められます。

例えば：
- 自分の1× 5つ星 + フレンド = 5 SE
- 自分の1× 5つ星 + 1× 3つ星ヒーラー + フレンド = 8 SE
- 自分の4× 5つ星 + フレンド = 18 SE（最大）

自分のカードが3枚になったら、4枚目を追加するのは比較的安い - 15 SEから18 SEへは5つ星でも+3だけなので、最強のフルチームを使っても良いでしょう。

**覚えておいてください：** カードドロップフロアはHPが2倍にジャンプしてから戻ることが多いです。生存力を評価し、最終的にはヒーラーを追加するか、段階的にダメージを追加する必要があります。

### どこまで最適化する？

SEコストをどこまで最適化するかは自分で決められますが、ほとんどのプレイヤーにとっては、3つ星や4つ星だけでどこまで生存できるか試すより、最高の5つ星1枚から始めるのが簡単です。その後、直面する課題に基づいて調整しましょう。
