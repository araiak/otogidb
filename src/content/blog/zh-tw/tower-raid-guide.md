這是一份面向新手（或相對較新的）玩家的塔樓副本活動指南。塔樓活動對新玩家非常友好，因為你可以向公會成員尋求幫助——如果你找到一個友善的公會，很有可能在幫助下獲得多張5星卡牌。

在塔樓活動中，你可以透過進行普通內容或使用SE搜索BOSS來遇到副本BOSS。

---

## 快速入門清單

### 活動前
1. **找一個活躍且樂於助人的公會** - 這對在困難樓層獲得幫助至關重要
2. **檢查你的好友列表**是否有 :card[740] 助戰 - 如有需要添加幻靈玩家（或者如果你更喜歡神靈/獸靈配置也可以）
3. **從下方列表中選擇你的坦克**（或使用你擁有的任何耐久卡牌）
4. **填滿你的後備位**，放入你擁有的傷害輔助或團隊增益卡
5. **如果你需要坦克**，去拍賣行查找

### 活動中
1. **配置：**借用的主C + 1張你的耐久卡（3-5 SE）
2. **堅持到計時器結束** → 分享給公會尋求幫助
3. **團滅了？** 不要復活（除非是掉落卡牌的樓層）
4. **遇到瓶頸？** 添加更多自己的卡牌或升級你的坦克以便能堅持到計時器結束後分享

## 塔樓活動特點

塔樓活動的特點：

- **多回合**，因此幾乎所有技能和能力都有用：
  - *最終波次能力* 在戰鬥的最後一波激活
  - *累積增益* 顯示「在波次開始時獲得X」 - 這些會在多回合中疊加
  - *固定增益* 始終生效
- **無屬性**，所以無論元素如何都使用你最強的隊伍
- **可變SE消耗**（根據你的卡牌，3-18 SE）
- **可繼續** - 如果時間到了，你可以花費SE+1繼續造成傷害，或將BOSS分享給公會尋求幫助。例如，如果你的隊伍消耗18 SE，你可以花19 SE再次造成相同的傷害。
- **經驗和掉落率加成無效** - 留給故事活動用吧！

### SE消耗計算

**重要：**只有你自己的卡牌計入SE消耗。借用的好友和後備位是免費的！

SE消耗 = 你的1-4張卡牌的星級總和（最小3，最大18）

- 你的1張3星 = 3 SE
- 你的1張5星 = 5 SE
- 你的2張5星 = 10 SE
- 借用好友的卡牌 = 免費
- 後備位 = 免費

這意味著你可以用一張強力的借用主C搭配一張自己的低星卡牌，只需很少的SE。

## 低SE爬塔策略

由於借用的助戰是免費的，最佳策略是借用一個強力傷害主C，自己只出1張卡。你可以只用1張自己的卡盡可能往上爬——3星只需3 SE，或者5星需要5 SE。

**一定要填滿你的後備位！**後備位不計入SE消耗，所以塞滿強力輔助和支援能力。

**注意：**下面的卡牌只是大多數玩家合理能獲得的例子——有很多卡牌都可以用！這些只是一些容易獲得的選擇供參考。

### 投資優先級

對於沒有奶媽的戰鬥，增加傷害非常重要——清得越快，受到的傷害越少。你的優先級應該是：

1. **獲得一個足夠耐久的坦克** - 購買足夠的副本來解鎖關鍵坦克能力（等級見下方）
2. **投資後備位和輔助** - 傷害加成和減傷輔助比進一步升級你的坦克更有效果
3. **最後再滿破坦克** - 滿破很好但沒有強力後備位重要

**麻糬優化技巧：**注意解鎖關鍵能力所需的最低等級。對於後備位或輔助卡牌，一旦能力解鎖，你可以封印它們賣掉來回收一部分投資。

### 幻靈配置（推薦）

這是最容易獲得的配置，因為 :card[740] 作為好友助戰廣泛可用，而且幾張耐久幻靈卡很容易獲得。

**借用：** :card[740] - 廣泛可從好友處獲得，高攻擊力，高技能傷害，其累積傷害加成也會影響你的卡牌。

**你的卡牌（按優先級排序）：**

1. :card[738] (Lv. 80) - 支援能力顯著提升攻擊力
2. :card[227] (Lv. 70) - 良好的自我維持
3. :card[1489] (Lv. 80) - 與借用的酒吞有技能傷害協同，低血量時有自我維持
4. :card[1436] (Lv. 75) - 耐久的近期活動卡，容易獲得
5. :card[267] (Lv. 75) - 來自拍賣行，戰鬥時間對較弱的隊伍有幫助，良好的有效生命值（注意：如果你想堅持到分享給公會，額外的戰鬥時間可能會對你不利）
6. :card[687] (Lv. 80) - 來自拍賣行的進攻選項，但可能更早需要減傷輔助或奶媽

### 神靈配置

同樣的策略適用於神靈，但坦克缺乏減傷，所以在高層推薦減傷輔助，或者你可能更早需要奶媽。

**借用：** :card[1311]、:card[739] 或 :card[553]

**你的卡牌（按優先級排序）：**

1. :card[542] (Lv. 80) - 來自拍賣行，更高的有效生命值
2. :card[358] (Lv. 70) - 較老所以拍賣行庫存較少，但有效生命值不錯
3. :card[593] (Lv. 70) - 來自拍賣行
4. :card[6] (Lv. 70) - 相當弱，但所有玩家都能從故事模式獲得一份

### 獸靈配置

獸靈的可獲得選項較少，主C在好友列表中也不太常見——你可能需要專門添加獸靈玩家。

**借用：** :card[674]、:card[630]、:card[543] 或 :card[514]

**你的卡牌（按優先級排序）：**

1. :card[586] (Lv. 75) - 來自拍賣行但較老/庫存少
2. :card[722] (Lv. 70) - 6個月前的活動，不是很耐久所以可能需要減傷輔助
3. :card[670] (Lv. 75) - 來自拍賣行的極高有效生命值4星卡，但傷害貢獻低

### 後備位輔助

由於後備位是免費的，最大化它們的價值：

- :card[664] (Lv. 85) - 2張卡傷害加成
- :card[618] (Lv. 85) - 比 :card[664] 弱的傷害選項
- :card[723] (Lv. 75) - 多目標傷害
- :card[643] (Lv. 75) - 多目標傷害
- :card[656] (Lv. 75+) - 僅限獸靈的傷害加成
- :card[688] (Lv. 70+) - 為有攻擊技能的近戰提供平衡的傷害/防禦

任何多目標傷害加成都可以。即使是只增益你主C的單目標輔助也可以。

**暴擊傷害加成**在你後備位有 :card[1307] 或 :card[1494] 時特別強力。

**減傷輔助**幫助你的坦克在高層生存。例如：:card[1062]、:card[688]、:card[1437]、:card[749]、:card[657]、:card[677]

### 後備位怪物（非輔助）

拍賣行的強力後備位怪物非常昂貴，但一些選項包括：
- :card[642] (Lv. 90) - 只有滿破才真正有效
- :card[672] (Lv. 90) - 只有滿破才有效
- :card[228] (Lv. 75) - 僅限獸靈

你的大部分後備位可能不會來自拍賣行，所以檢查你有什麼對隊伍有益的卡牌。一些值得尋找的標準抽卡池卡牌：

- :card[2373] (Lv. 75) - 僅限幻靈
- :card[2191] (Lv. 75) - 僅限獸靈
- :card[1321] (Lv. 75+) - 神靈
- :card[744] (Lv. 85) - 任何屬性，每回合15%傷害非常巨大！可用於所有多回合內容（塔樓副本、故事）作為後備，但不能用於單回合。如果你能達到85級值得使用PvP寶珠（之前無效）——否則為世界BOSS或PvP等單回合內容節省寶珠
- :card[661] (Lv. 85) - 任何屬性
- :card[600] (Lv. 77) - 4星但可能值得使用PvP寶珠，40%暴擊傷害是很大的加成

你可以在卡牌列表上使用「team」篩選器並與你的收藏對比，找到具有全隊能力的怪物完整列表。

## 擴展你的隊伍

一旦遇到瓶頸，逐步添加你自己的卡牌到隊伍中。你始終有1個借用的好友，然後你的SE消耗是你的1-4張卡牌的星級總和（最小3，最大18）。

**你的第二張卡應該是奶媽** - 這能幫助你堅持到計時器結束，這樣你就可以分享給公會尋求幫助。

例如：
- 你的1張5星 + 好友 = 5 SE
- 你的1張5星 + 1張3星奶媽 + 好友 = 8 SE
- 你的4張5星 + 好友 = 18 SE（最大）

一旦你使用3張自己的卡牌，添加第4張相對便宜——從15 SE到18 SE只增加3點就能再加一張5星，所以你可以直接使用你最強的完整隊伍。

**記住：**掉落卡牌的樓層通常會跳到2倍血量然後再降回來。你需要評估你的生存能力，最終添加奶媽，或逐步添加更多傷害。

### 要優化到什麼程度？

你可以決定要對SE消耗優化到什麼程度，但對大多數玩家來說，從你最好的1張5星開始比嘗試看你只用3星或4星卡能走多遠更容易。然後根據你遇到的挑戰進行調整。

## 總結

| 情況 | 策略 |
|------|------|
| 剛開始 | 借用強力主C，提供1張你的耐久卡（3-5 SE）|
| 需要更多生存 | 使用更耐久的卡或添加減傷輔助 |
| 遇到瓶頸 | 將BOSS分享給公會尋求幫助，或添加更多你自己的卡 |
| 掉落卡牌樓層（2倍血量）| 考慮臨時添加卡牌應對血量峰值 |
| 後期塔樓 | 使用你最強的完整隊伍（最多4張卡 + 好友）|
