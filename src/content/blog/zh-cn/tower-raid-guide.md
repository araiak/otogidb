这是一份面向新手（或相对较新的）玩家的塔楼副本活动指南。塔楼活动对新玩家非常友好，因为你可以向公会成员寻求帮助——如果你找到一个友善的公会，很有可能在帮助下获得多张5星卡牌。

在塔楼活动中，你可以通过进行普通内容或使用SE搜索BOSS来遇到副本BOSS。

---

## 快速入门清单

### 活动前
1. **找一个活跃且乐于助人的公会** - 这对在困难楼层获得帮助至关重要
2. **检查你的好友列表**是否有 :card[740] 助战 - 如有需要添加幻灵玩家（或者如果你更喜欢神灵/兽灵配置也可以）
3. **从下方列表中选择你的坦克**（或使用你拥有的任何耐久卡牌）
4. **填满你的后备位**，放入你拥有的伤害辅助或团队增益卡
5. **如果你需要坦克**，去拍卖行查找

### 活动中
1. **配置：**借用的主C + 1张你的耐久卡（3-5 SE）
2. **坚持到计时器结束** → 分享给公会寻求帮助
3. **团灭了？** 不要复活（除非是掉落卡牌的楼层）
4. **遇到瓶颈？** 添加更多自己的卡牌或升级你的坦克以便能坚持到计时器结束后分享

## 塔楼活动特点

塔楼活动的特点：

- **多回合**，因此几乎所有技能和能力都有用：
  - *最终波次能力* 在战斗的最后一波激活
  - *累积增益* 显示"在波次开始时获得X" - 这些会在多回合中叠加
  - *固定增益* 始终生效
- **无属性**，所以无论元素如何都使用你最强的队伍
- **可变SE消耗**（根据你的卡牌，3-18 SE）
- **可继续** - 如果时间到了，你可以花费SE+1继续造成伤害，或将BOSS分享给公会寻求帮助。例如，如果你的队伍消耗18 SE，你可以花19 SE再次造成相同的伤害。
- **经验和掉落率加成无效** - 留给故事活动用吧！

### SE消耗计算

**重要：**只有你自己的卡牌计入SE消耗。借用的好友和后备位是免费的！

SE消耗 = 你的1-4张卡牌的星级总和（最小3，最大18）

- 你的1张3星 = 3 SE
- 你的1张5星 = 5 SE
- 你的2张5星 = 10 SE
- 借用好友的卡牌 = 免费
- 后备位 = 免费

这意味着你可以用一张强力的借用主C搭配一张自己的低星卡牌，只需很少的SE。

## 低SE爬塔策略

由于借用的助战是免费的，最佳策略是借用一个强力伤害主C，自己只出1张卡。你可以只用1张自己的卡尽可能往上爬——3星只需3 SE，或者5星需要5 SE。

**一定要填满你的后备位！**后备位不计入SE消耗，所以塞满强力辅助和支援能力。

**注意：**下面的卡牌只是大多数玩家合理能获得的例子——有很多卡牌都可以用！这些只是一些容易获得的选择供参考。

### 投资优先级

对于没有奶妈的战斗，增加伤害非常重要——清得越快，受到的伤害越少。你的优先级应该是：

1. **获得一个足够耐久的坦克** - 购买足够的副本来解锁关键坦克能力（等级见下方）
2. **投资后备位和辅助** - 伤害加成和减伤辅助比进一步升级你的坦克更有效果
3. **最后再满破坦克** - 满破很好但没有强力后备位重要

**麻糬优化技巧：**注意解锁关键能力所需的最低等级。对于后备位或辅助卡牌，一旦能力解锁，你可以封印它们卖掉来回收一部分投资。

### 幻灵配置（推荐）

这是最容易获得的配置，因为 :card[740] 作为好友助战广泛可用，而且几张耐久幻灵卡很容易获得。

**借用：** :card[740] - 广泛可从好友处获得，高攻击力，高技能伤害，其累积伤害加成也会影响你的卡牌。

**你的卡牌（按优先级排序）：**

1. :card[738] (Lv. 80) - 支援能力显著提升攻击力
2. :card[227] (Lv. 70) - 良好的自我维持
3. :card[1489] (Lv. 80) - 与借用的酒吞有技能伤害协同，低血量时有自我维持
4. :card[1436] (Lv. 75) - 耐久的近期活动卡，容易获得
5. :card[267] (Lv. 75) - 来自拍卖行，战斗时间对较弱的队伍有帮助，良好的有效生命值（注意：如果你想坚持到分享给公会，额外的战斗时间可能会对你不利）
6. :card[687] (Lv. 80) - 来自拍卖行的进攻选项，但可能更早需要减伤辅助或奶妈

### 神灵配置

同样的策略适用于神灵，但坦克缺乏减伤，所以在高层推荐减伤辅助，或者你可能更早需要奶妈。

**借用：** :card[1311] 或 :card[553]

**你的卡牌（按优先级排序）：**

1. :card[542] (Lv. 80) - 来自拍卖行，更高的有效生命值
2. :card[358] (Lv. 70) - 较老所以拍卖行库存较少，但有效生命值不错
3. :card[593] (Lv. 70) - 来自拍卖行
4. :card[6] (Lv. 70) - 相当弱，但所有玩家都能从故事模式获得一份

### 兽灵配置

兽灵的可获得选项较少，主C在好友列表中也不太常见——你可能需要专门添加兽灵玩家。

**借用：** :card[630]、:card[543] 或 :card[514]

**你的卡牌（按优先级排序）：**

1. :card[586] (Lv. 75) - 来自拍卖行但较老/库存少
2. :card[722] (Lv. 70) - 6个月前的活动，不是很耐久所以可能需要减伤辅助
3. :card[670] (Lv. 75) - 来自拍卖行的极高有效生命值4星卡，但伤害贡献低

### 后备位辅助

由于后备位是免费的，最大化它们的价值：

- :card[664] (Lv. 85) - 2张卡伤害加成
- :card[618] (Lv. 85) - 比 :card[664] 弱的伤害选项
- :card[723] (Lv. 75) - 多目标伤害
- :card[643] (Lv. 75) - 多目标伤害
- :card[656] (Lv. 75+) - 仅限兽灵的伤害加成
任何多目标伤害加成都可以。即使是只增益你主C的单目标辅助也可以。

**暴击伤害加成**在你后备位有 :card[1307] 或 :card[1494] 时特别强力。

**减伤辅助**帮助你的坦克在高层生存。例如：:card[1062]、:card[1437]、:card[749]、:card[657]、:card[677]

### 后备位怪物（非辅助）

拍卖行的强力后备位怪物非常昂贵，但一些选项包括：
- :card[642] (Lv. 90) - 只有满破才真正有效
- :card[672] (Lv. 90) - 只有满破才有效
- :card[228] (Lv. 75) - 仅限兽灵

你的大部分后备位可能不会来自拍卖行，所以检查你有什么对队伍有益的卡牌。一些值得寻找的标准抽卡池卡牌：

- :card[2373] (Lv. 75) - 仅限幻灵
- :card[2191] (Lv. 75) - 仅限兽灵
- :card[1321] (Lv. 75+) - 神灵
- :card[744] (Lv. 85) - 任何属性，每回合15%伤害非常巨大！可用于所有多回合内容（塔楼副本、故事）作为后备，但不能用于单回合。如果你能达到85级值得使用PvP宝珠（之前无效）——否则为世界BOSS或PvP等单回合内容节省宝珠
- :card[661] (Lv. 85) - 任何属性
- :card[600] (Lv. 77) - 4星但可能值得使用PvP宝珠，40%暴击伤害是很大的加成

你可以在卡牌列表上使用"team"筛选器并与你的收藏对比，找到具有全队能力的怪物完整列表。

## 扩展你的队伍

一旦遇到瓶颈，逐步添加你自己的卡牌到队伍中。你始终有1个借用的好友，然后你的SE消耗是你的1-4张卡牌的星级总和（最小3，最大18）。

**你的第二张卡应该是奶妈** - 这能帮助你坚持到计时器结束，这样你就可以分享给公会寻求帮助。

例如：
- 你的1张5星 + 好友 = 5 SE
- 你的1张5星 + 1张3星奶妈 + 好友 = 8 SE
- 你的4张5星 + 好友 = 18 SE（最大）

一旦你使用3张自己的卡牌，添加第4张相对便宜——从15 SE到18 SE只增加3点就能再加一张5星，所以你可以直接使用你最强的完整队伍。

**记住：**掉落卡牌的楼层通常会跳到2倍血量然后再降回来。你需要评估你的生存能力，最终添加奶妈，或逐步添加更多伤害。

### 要优化到什么程度？

你可以决定要对SE消耗优化到什么程度，但对大多数玩家来说，从你最好的1张5星开始比尝试看你只用3星或4星卡能走多远更容易。然后根据你遇到的挑战进行调整。

## 总结

| 情况 | 策略 |
|------|------|
| 刚开始 | 借用强力主C，提供1张你的耐久卡（3-5 SE）|
| 需要更多生存 | 使用更耐久的卡或添加减伤辅助 |
| 遇到瓶颈 | 将BOSS分享给公会寻求帮助，或添加更多你自己的卡 |
| 掉落卡牌楼层（2倍血量）| 考虑临时添加卡牌应对血量峰值 |
| 后期塔楼 | 使用你最强的完整队伍（最多4张卡 + 好友）|
