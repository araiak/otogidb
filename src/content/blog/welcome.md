# Welcome to OtogiDB

OtogiDB is a comprehensive card database for **Otogi Spirit Agents**, featuring detailed stats, skills, and abilities for all cards in the game.

## Why OtogiDB?

Otogi Spirit Agents was re-released in 2024, and many online resources still contain outdated information from the original release. Card stats, skill descriptions, and game mechanics have changed significantly since then.

**OtogiDB provides accurate, up-to-date data** extracted directly from the current game client, ensuring you have reliable information for team building and strategy.

## Features

- **Complete Card Database**: Browse all 868+ cards with full stats
- **Search & Filter**: Find cards by name, skill, attribute, or type
- **Similar Cards**: Find replacement options based on attribute, type, and skill tags
- **Update History**: Track card changes across game versions
- **Mobile Friendly**: Works great on phones and tablets

## Using Card References

In blog posts, you can reference cards using the `:card[id]` syntax. For example:

- :card[1] - Reference by ID
- :card[Nue] - Reference by name

Hover over these links to see a preview of the card's stats and skills.

## Contributing

OtogiDB is open source! We welcome contributions from the community:

- **Blog Posts**: Share guides, tier lists, team compositions, or game mechanics explanations
- **Bug Reports**: Found incorrect data or a website issue? Let us know
- **Feature Requests**: Have ideas to improve the site?

To contribute a blog post, simply create a Markdown file in the `src/content/blog/` folder and submit a pull request to our [GitHub repository](https://github.com/araiak/otogidb).

### Blog Post Format

```markdown
---
title: "Your Post Title"
description: "A brief description"
date: "2025-01-01"
author: "Your Name"
tags: ["guide", "team-building"]
---

Your content here. Use :card[Nue] to reference cards!
```

## Getting Started

Head to the [Cards](/) page to browse the full database, or check out the [Updates](/updates) page to see recent game changes.

Have questions or feedback? Open an issue on [GitHub](https://github.com/araiak/otogidb/issues).

# OtogiDBへようこそ

OtogiDBは**あやかし百鬼夜行**の総合カードデータベースです。ゲーム内の全カードの詳細なステータス、スキル、アビリティを掲載しています。

## なぜOtogiDB？

あやかし百鬼夜行は2024年に再リリースされましたが、多くのオンラインリソースには初版からの古い情報が残っています。カードのステータス、スキルの説明、ゲームの仕組みは大きく変更されています。

**OtogiDBは正確で最新のデータを提供します** - 現在のゲームクライアントから直接抽出したデータなので、チーム編成や戦略立案に信頼できる情報を得られます。

## 機能

- **完全なカードデータベース**: 868枚以上のカードを完全なステータスで閲覧
- **検索とフィルター**: 名前、スキル、属性、タイプでカードを検索
- **類似カード**: 属性、タイプ、スキルタグに基づいた代替カードを発見
- **更新履歴**: ゲームバージョン間のカード変更を追跡
- **モバイル対応**: スマートフォンやタブレットでも快適に使用可能

## カード参照の使い方

ブログ記事では `:card[id]` 構文でカードを参照できます。例えば：

- :card[1] - IDで参照
- :card[Nue] - 名前で参照

これらのリンクにカーソルを合わせると、カードのステータスとスキルのプレビューが表示されます。

## 貢献

OtogiDBはオープンソースです！コミュニティからの貢献を歓迎します：

- **ブログ記事**: ガイド、ティアリスト、チーム編成、ゲームの仕組みの解説を共有
- **バグ報告**: 不正確なデータやウェブサイトの問題を発見したらお知らせください
- **機能リクエスト**: サイトを改善するアイデアがありますか？

ブログ記事を投稿するには、`src/content/blog/` フォルダにMarkdownファイルを作成し、[GitHubリポジトリ](https://github.com/araiak/otogidb)にプルリクエストを送信してください。

### ブログ記事のフォーマット

```markdown

title: "記事タイトル"
description: "簡単な説明"
date: "2025-01-01"
author: "あなたの名前"
tags: ["ガイド", "チーム編成"]
locale: "ja"
---

ここに内容を書きます。:card[Nue] でカードを参照できます！
```

[カード](/)ページで完全なデータベースを閲覧するか、[アップデート](/ja/updates)ページで最近のゲーム変更を確認してください。

# OtogiDB에 오신 것을 환영합니다

OtogiDB는 **오토기 스피릿 에이전츠**의 종합 카드 데이터베이스로, 게임 내 모든 카드의 상세한 스탯, 스킬, 어빌리티를 제공합니다.

## 왜 OtogiDB인가요?

오토기 스피릿 에이전츠는 2024년에 재출시되었지만, 많은 온라인 자료에는 원래 버전의 오래된 정보가 남아 있습니다. 카드 스탯, 스킬 설명, 게임 메커니즘이 크게 변경되었습니다.

**OtogiDB는 정확하고 최신의 데이터를 제공합니다** - 현재 게임 클라이언트에서 직접 추출한 데이터로, 팀 구성과 전략에 신뢰할 수 있는 정보를 얻을 수 있습니다.

## 기능

- **완전한 카드 데이터베이스**: 868장 이상의 카드를 전체 스탯과 함께 확인
- **검색 및 필터**: 이름, 스킬, 속성, 타입으로 카드 검색
- **유사 카드**: 속성, 타입, 스킬 태그 기반 대체 카드 찾기
- **업데이트 기록**: 게임 버전 간 카드 변경 사항 추적
- **모바일 친화적**: 스마트폰과 태블릿에서도 쾌적하게 사용

## 카드 참조 사용법

블로그 게시물에서 `:card[id]` 구문으로 카드를 참조할 수 있습니다:

- :card[1] - ID로 참조
- :card[Nue] - 이름으로 참조

이 링크에 마우스를 올리면 카드의 스탯과 스킬 미리보기가 표시됩니다.

## 기여하기

OtogiDB는 오픈 소스입니다! 커뮤니티의 기여를 환영합니다:

- **블로그 게시물**: 가이드, 티어 리스트, 팀 구성, 게임 메커니즘 설명 공유
- **버그 리포트**: 부정확한 데이터나 웹사이트 문제를 발견하면 알려주세요
- **기능 요청**: 사이트 개선 아이디어가 있으신가요?

블로그 게시물을 기여하려면 `src/content/blog/` 폴더에 마크다운 파일을 생성하고 [GitHub 저장소](https://github.com/araiak/otogidb)에 풀 리퀘스트를 제출하세요.

## 시작하기

[카드](/)페이지에서 전체 데이터베이스를 둘러보거나, [업데이트](/ko/updates) 페이지에서 최근 게임 변경 사항을 확인하세요.

질문이나 피드백이 있으신가요? [GitHub](https://github.com/araiak/otogidb/issues)에서 이슈를 열어주세요.

# Bienvenido a OtogiDB

OtogiDB es una base de datos completa de cartas para **Otogi Spirit Agents**, con estadísticas detalladas, habilidades y capacidades de todas las cartas del juego.

## ¿Por qué OtogiDB?

Otogi Spirit Agents se relanzó en 2024, y muchos recursos en línea todavía contienen información desactualizada de la versión original. Las estadísticas de las cartas, descripciones de habilidades y mecánicas del juego han cambiado significativamente.

**OtogiDB proporciona datos precisos y actualizados** extraídos directamente del cliente del juego actual, asegurando información confiable para la construcción de equipos y estrategias.

## Características

- **Base de datos completa**: Explora más de 868 cartas con estadísticas completas
- **Búsqueda y filtros**: Encuentra cartas por nombre, habilidad, atributo o tipo
- **Cartas similares**: Encuentra opciones de reemplazo basadas en atributo, tipo y etiquetas de habilidad
- **Historial de actualizaciones**: Rastrea cambios de cartas entre versiones
- **Compatible con móviles**: Funciona perfectamente en teléfonos y tabletas

## Uso de referencias de cartas

En las publicaciones del blog, puedes hacer referencia a cartas usando la sintaxis `:card[id]`:

- :card[1] - Referencia por ID
- :card[Nue] - Referencia por nombre

Pasa el cursor sobre estos enlaces para ver una vista previa de las estadísticas y habilidades de la carta.

## Contribuir

¡OtogiDB es de código abierto! Damos la bienvenida a las contribuciones de la comunidad:

- **Publicaciones del blog**: Comparte guías, listas de niveles, composiciones de equipos o explicaciones de mecánicas
- **Reportes de errores**: ¿Encontraste datos incorrectos o un problema en el sitio? Avísanos
- **Solicitudes de funciones**: ¿Tienes ideas para mejorar el sitio?

Para contribuir una publicación, crea un archivo Markdown en la carpeta `src/content/blog/` y envía un pull request a nuestro [repositorio de GitHub](https://github.com/araiak/otogidb).

## Comenzar

Dirígete a la página de [Cartas](/) para explorar la base de datos completa, o consulta la página de [Actualizaciones](/es/updates) para ver los cambios recientes del juego.

¿Tienes preguntas o comentarios? Abre un issue en [GitHub](https://github.com/araiak/otogidb/issues).
